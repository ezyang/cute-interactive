# CuTe Layout Interactive Notebooks (marimo)

marimo versions of the CuTe Layout interactive notebooks. Each notebook illustrates examples from the CuTe Layout paper using the `tensor-layouts` Python library.

All notebooks are WASM-compatible and can be run directly in the browser.
